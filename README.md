# Weather Application 
A simple React weather application that shows current weather details using the OpenWeather API.

## Features
- Search weather by city name
- Displays temperature, humidity & weather condition
- Responsive UI for mobile and desktop

### Search History
- Stores the last 5 searched cities
- Recent searches shown as clickable buttons
- Click to search the city again instantly
- Clear history option available
- Uses localStorage to persist data

## Tech Stack
- React
- Tailwind CSS
- OpenWeather API

## Run
npm install
npm run dev
