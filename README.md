# Interport-Global-Logistics-Pvt-Ltd-Weather-Application-
A React weather app that fetches live weather data by city.

## Features
- Search weather by city name
- Displays temperature, feels-like temperature humidity & weather icons
- Responsive UI for mobile and desktop

### Search History
- Stores the last 5 searched cities
- Recent searches shown as clickable buttons
- Click to search the city again instantly
- Clear history option available
- Uses localStorage to persist data

## Tech Stack
- React
- Tailwind CSS
- OpenWeather API

## Run
npm install
npm run dev
